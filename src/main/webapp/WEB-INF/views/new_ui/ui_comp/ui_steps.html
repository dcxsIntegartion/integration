steps操作说明

  • 本组件基于jquery-steps插件
  • 本例当中使用的设置：headerTag: h3，bodyTag: "section"就是将#steps_demo里的h3作为每个步骤的标签，将section作为每个步骤的内容
  • 内容当中需要js控制的部分，必须写在$("#steps_demo").steps({})执行完成后面
